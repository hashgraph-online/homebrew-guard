# homebrew-tap

Homebrew tap for [HOL Guard](https://hol.org/guard) by [Hashgraph Online](https://github.com/hashgraph-online).

## Install

```bash
brew tap hashgraph-online/tap
brew install hol-guard
```

Then initialize:

```bash
hol-guard init
```

## What is HOL Guard?

HOL Guard is a local-first security harness that protects AI coding agents (Codex, Claude Code, Cursor, Gemini, Hermes, and more) by intercepting tool actions before files change or network is contacted.

- [PyPI](https://pypi.org/project/hol-guard/)
- [GitHub](https://github.com/hashgraph-online/hol-guard)
- [Docs](https://hol.org/guard)
